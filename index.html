CODE 243
Damaged Switch
Definition
Use code 243 if any part of the switch is damaged or broken, including insulators, switchblades, and other components.
Key Points
01
Includes insulators, switchblades, and all mechanical parts
02
Does not require on-site maintenance response
03
Enter a detailed description in the comments section of your MDT iPad
No On-Site Maintenance Required
